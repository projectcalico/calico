Project Calico is an open-source Networking project for cloud-native applications

Calico enables secure IP communication between containers, virtual machines or bare metal workloads.

Secure

Designed from the ground up to support rich, flexible and secure network policy.

Scalable

Simple

Easy to deploy, easy to operate, easy to diagnose.

Open

Open source and open standards.

Any Workload

Supports container, VM and bare metal workloads.
